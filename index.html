My Bookmarks

This site is to create a list of websites that can be rated, described, and accessed by anyone who visits the page. Simply filling in the form at the top will add a website to the list below. Once in the list a bookmark can be clicked on to expand it's view, giving the visitor the option to edit the description and the star rating of a site, as well as the option to remove it entirely and a link to visit the page being bookmarked in the list.

This is a simple site mainly to be used to demonstrate RESTful API practices and the use of jQuery. It is not intended for live use.
